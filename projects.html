Web Scraping

In May and June of 2024 as part of my work with JMA Consulting, I updated or rewrote 78 scrapers for Open North Represent's opencivicdata project. These scrapers get the name, role, ward, and contact information for each member of the municipal council for a municpality and store them in a database. These scrapers were built using pupa, a library that Open North created for building scrapers, as well as the more generel python libraries urllib3, requests, and json, among others. Another contribution I made to the project was that I implemented a function to find cloudflare protected emails on these sites and decode them from their protected format into a normal email. You can view the source code for the scrapers here on Github.
